Course Overview

This cross-disciplinary course explores the multifaceted nature of persuasion in the modern world, bridging political science, communication, journalism, marketing, and public health. The course combines rich theories of communication with practical, hands-on analysis.

Students will engage with real-world data using statistical and computational methods, including:

  • Randomized controlled trials
  • Natural experiments
  • Machine learning techniques
  • Causal inference methods
Course Requirements

Four Short Assignments (70%)

  1. Comparison of Self-Reported Media Consumption and Behavioral Data

    Analysis of personal media consumption patterns using self-assessment and behavioral data

  2. Analyzing Web-Tracking Data

    Application of causal inference techniques to web tracking data

  3. Extension of Experimental Data

    Replication and extension analysis using Time-sharing Experiments for the Social Sciences

  4. Design Your Own Persuasion Experiment

    Creation and evaluation of original media content for behavioral influence

Final exam will constitute the remaining 30% of the grade.
